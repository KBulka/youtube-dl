# YouTube Auto-Downloader

Automatyczne pobieranie filmów z YouTube po skopiowaniu linku do schowka.

## Szybki start

1. **Uruchom auto-downloader:**
   ```
   Kliknij dwukrotnie: start_auto_downloader.bat
   ```

2. **Skopiuj link YouTube** (Ctrl+C)

3. **Film pobierze się automatycznie!**

## Lokalizacja pobranych filmów

```
C:\Users\Bulka\Downloads\YouTube\
```

## Konfiguracja

Edytuj `config.json` aby zmienić:
- Folder pobierania
- Format wideo
- Powiadomienia
- Szablon nazwy pliku

## Obsługiwane linki

✅ `https://www.youtube.com/watch?v=...`  
✅ `https://youtu.be/...`  
✅ `https://www.youtube.com/shorts/...`  
✅ Playlisty YouTube

## Zatrzymanie

Naciśnij **Ctrl+C** w oknie konsoli
